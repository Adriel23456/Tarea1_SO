================================================================
                   IMAGE PROCESSING CLIENT
================================================================

Version: 1.0.0
Build Date: 2024

Developed for Systems Operations Course

================================================================
                        FEATURES
================================================================

• Advanced Image Processing
  - Histogram Equalization for contrast enhancement
  - Automatic color-based classification
  - Support for JPG, JPEG, PNG, and GIF formats

• Network Communication
  - HTTP/TCP protocol support
  - Configurable server connection
  - Automatic retry mechanism
  - Chunked file transfer

• Modern User Interface
  - Built with GTK4 framework
  - Responsive and intuitive design
  - Real-time status updates
  - Efficient file management

================================================================
                     TECHNOLOGIES
================================================================

• Programming Language: C
• GUI Framework: GTK4
• Network Protocol: HTTP/TCP
• Build System: GNU Make
• Platform: Linux (SystemD/SysVinit compatible)

================================================================
                    ARCHITECTURE
================================================================

• Client-Server Model
• Multithreaded Processing
• Priority Queue Implementation
• Daemon Service Integration
